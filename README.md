# Sistema de Reservas de Canchas

Aplicación de consola en Java para gestionar reservas de canchas deportivas.
Evita conflictos de horario y permite guardar y cargar la información entre sesiones.

Proyecto desarrollado para aprender Programación Orientada a Objetos en Java.

## Funcionalidades

- Crear canchas deportivas
- Reservar horarios para una cancha
- Validar disponibilidad (sin conflictos de horario)
- Guardar y cargar reservas desde archivos
- Manejo de horarios con LocalTime

## Tecnologías

![Java](https://img.shields.io/badge/Java-ED8B00?style=for-the-badge&logo=openjdk&logoColor=white)

- Java 17+
- Programación Orientada a Objetos (POO)
- Manejo de archivos para persistencia de datos
- LocalTime para gestión de horarios

## Cómo ejecutar

1. Clona el repositorio:
```bash
git clone https://github.com/JCastellanosDev/SistemaReservasCanchas.git
```

2. Ábrelo en IntelliJ IDEA o tu editor favorito

3. Ejecuta la clase `Main.java`
